Following dependencies are required in order to run the application (in project's lib dir):
commons-codec-1.3.jar
commons-logging-1.1.1.jar
google-api-client-1.4.1-beta.jar 
google-api-client-googleapis-1.4.1-beta.jar 
google-api-services-latitude-v1-1.1.0-beta.jar 
gson-1.6.jar 
guava-r09.jar 
httpclient-4.0.3.jar 
httpcore-4.0.1.jar 
jackson-core-asl-1.6.7.jar

ALTitude also uses:
- google AdMob
- ActionBarSherlock by Jake Wharton and can be obtained here: https://github.com/JakeWharton/ActionBarSherlock
